Wheel - Spin is a lightweight, mobile-friendly “spin the wheel” name picker built as a single-page web app and hosted with GitHub Pages. It’s designed to be easy for non-technical users: open the link and spin, with no setup required.
-------------------------------------------

Features:

Max 8 names (wheel automatically scales to the number of filled names).

Tap the center (“SNURRA”) to spin (plus a regular Spin button).

Swedish interface for simple everyday use.

Winner highlight + subtle confetti.

Optional sound (off by default).

Optional winner history (last 5 winners).

Optional “Lock editing” to prevent accidental changes.

Settings and name list are stored per device (no accounts, no cloud).
-------------------------------------------

Hosting This project is intended to be hosted on GitHub Pages, making it easy to share as a link or save as a home screen shortcut like an app.
